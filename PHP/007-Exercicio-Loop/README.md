Dicas para construir o código.
* Use IF ou Switch. // Alterar
* Obtenha o modulo da divisão com o operador **%** //alterar
* [Documentação PHP - Switch][doc-switch] // alterar
* [Documentação PHP - Mod][doc-mod] // alterar

```php
<?php
switch ($i) {
    case "apple":
        echo "i is apple";
        break;
    case "bar":
        echo "i is bar";
        break;
    case "cake":
        echo "i is cake";
        break;
} // alterar
?>
```
